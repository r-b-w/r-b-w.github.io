Programação Paralela

Em primeiro lugar, é necessário um bom conhecimento da linguagem Python para acompanhar esta disciplina.

Existe uma infinidade de tutoriais, em texto e em vídeo, qualquer um semidecente serve. O lugar óbvio de começar é por aqui. Não sigam as instruções de instalação, no entanto, vamos usar o Anaconda neste curso pois torna a vida muito mais fácil.

Se você não sabe nada de programação (e talvez não devesse estar estudando programação paralela ainda) e tem paciência, você pode assistir este curso.

Para quem tem um conhecimento razoável de alguma outra linguagem de programação, honestamente eu acho que o minitutorial oficial da linguagem é suficiente.

Se você quer uma coisa um pouco mais cadenciada, este aqui é bem razoável.

O material do curso está neste diretório.
